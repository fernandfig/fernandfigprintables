Free hosting options:

GitHub Pages
1) Create a public repo (any name).
2) Upload these files to the repo root.
3) Settings → Pages: Deploy from branch (main / root).
4) Add custom domain: fernandfigprintables.com and www.fernandfigprintables.com
5) DNS at your registrar:
   - A records: 185.199.108.153, 185.199.109.153, 185.199.110.153, 185.199.111.153
   - CNAME: www → yourusername.github.io

Netlify
1) Drag & drop this folder into Netlify.
2) Add your custom domain and follow DNS prompts for HTTPS.
3) Optional: connect GitHub for auto‑deploys.

Update Etsy links in index.html (two places) and contact email in contact.html.
